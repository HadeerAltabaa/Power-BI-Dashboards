# Data Jobs Dashboard — Power BI Project

![Data Jobs Dashboard](/images/Data%20Jobs%20Dashboard.png)

## Introduction

The **Data Jobs Dashboard** is a Power BI project designed to provide clear and structured insights into the data job market using a real-world dataset of **2024 data science job postings**.

It helps users understand key information about the job market, such as salaries, job demand, locations, and employment trends, in one interactive and easy-to-use dashboard.

This project is especially useful for:
- Job seekers  
- Job transitioners  
- Career switchers  

It solves a common problem where job market information is **scattered across multiple sources and difficult to analyze**.

## Dashboard Overview

### Page 1: High-Level Market Overview

![Data Jobs Dashboard](/images/Data%20Jobs%20Dashboard.png)

This page provides an overall view of the data job market.

It includes:

- Total number of job postings (KPI)
- Median yearly and hourly salaries
- Monthly job posting trends
- Salary comparison across job roles
- Most in-demand job titles

This page helps users quickly understand:
> The general state of the data job market and its trends

---

### Page 2: Job Title Drill Through

![Job Title Drill Through](/images/Job%20Title%20Drill%20Through.png)

This page provides detailed insights for each specific job title.

It includes:

- Salary distribution (yearly and hourly)
- Work-from-home percentage
- Health insurance availability
- Degree requirement analysis
- Job location distribution (world map)
- Job posting platforms comparison
- Employment type breakdown

This page helps users understand:
> The full breakdown of any selected job role in detail
